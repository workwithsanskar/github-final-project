# Simple Interest Calculator 💰

## 📌 Description
This project is a simple interest calculator that calculates the interest based on user input such as principal amount, rate of interest, and time period.

---

## 🚀 Features
- Takes user input (Principal, Rate, Time)
- Calculates simple interest
- Displays result instantly
- Beginner-friendly project

---

## 🧮 Formula Used
Simple Interest is calculated using:

SI = (P × R × T) / 100

Where:
- P = Principal
- R = Rate of Interest
- T = Time

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript / Bash (depending on implementation)

---

## 📷 Usage
1. Enter Principal amount
2. Enter Rate of Interest
3. Enter Time period
4. Click Calculate
5. View the result
